# 悖论餐厅 (Paradox Diner)

- Date: 2026-03-02
- Players: 2-6
- Time: 15-25 分钟
- Genre: 策略 / 时间管理 / 合作

## Elevator pitch
你是时空餐厅的服务员，要招待来自不同时代的客人。但小心——让维多利亚时代的绅士坐在未来赛博朋克玩家旁边？会引发时间悖论！

## Core loop
1. **抽取客人卡** —— 每位客人有：时代标记（古代/中世纪/维多利亚/现代/未来）、座位偏好、特殊需求
2. **安排座位** —— 将客人放在圆形餐桌的座位上，使用 🪑 表情符号可视化
3. **检查悖论** —— 相邻座位若时代冲突（如古代↔未来）则产生悖论点
4. **完成订单** —— 无悖论则得分，有悖论需立即解决（移动座位或牺牲分数）

## Rules (high level)
- 每轮出现 3-5 位新客人，必须在倒计时内（90秒）完成安排
- **时代兼容表**：相邻座位时代差距越大，悖论风险越高
  - 相邻时代（如现代↔未来）：安全
  - 跳一级（如维多利亚↔现代）：警告
  - 跳两级以上：悖论！
- **特殊客人效果**：
  - 时间旅行者：可以坐在任何时代旁边无惩罚
  - 时间警察：消除相邻所有悖论
  - 历史名人：若被安排在正确时代，双倍分数

## Win condition
- **合作模式**：累计达到目标分数（50/100/150 分三档难度）
- **竞争模式**：5轮后分数最高者获胜
- **完美通关**：无任何悖论完成一轮，获得"时间大师"称号

## Variants / twist
1. **时间风暴模式**：每3轮触发一次时间风暴，所有座位旋转随机位置，必须重新调整
2. **双人约会**：某些客人必须坐在一起，但如果他们来自敌对时代会产生额外悖论
3. **时间厨房**：加入订单准备环节，客人点的菜也有时代属性，必须与客人时代匹配
4. **倒放模式**：从满座开始，每轮移除客人，目标是让最后留下的客人时代完全兼容

---

*设计师注：灵感来自《Overcooked》的合作混乱 + 《Twilight Zone》的时间悖论概念*
