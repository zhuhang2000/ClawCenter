# 回声迷宫 (Echo Maze)

- Date: 2026-03-02
- Players: 4-8人
- Time: 20-40分钟
- Genre: 合作解谜 / 实时沟通

## Elevator pitch
一款考验"延迟沟通"的合作解谜游戏：探路者独自在隐形迷宫中探索，而指引者只能通过"回声"（延迟、碎片化、扭曲的信息）来传递方向。玩家必须在信息不完整的情况下，建立独特的"沟通协议"。

## Core loop
1. **迷宫生成**：系统随机生成一个看不见的网格迷宫（15×15）
2. **角色分配**：
   - 1名**探路者**（执行移动）
   - 其他人为**回声师**（只能观察迷宫地图）
3. **回声机制**：
   - 回声师可以发送信息，但有**3秒延迟**
   - 信息长度限制：最多5个字符（"上"、"右上"、"停"等）
   - 探路者移动后，**2秒内无法接收新信息**（模拟"回声盲区"）
4. **目标**：在60步内找到出口

## Rules (high level)
- 探路者**完全看不到迷宫**，只能依赖回声师的指引
- 回声师可以看到完整迷宫，但**无法直接控制**探路者
- 撞墙惩罚：探路者撞墙会**眩晕5秒**（无法移动/接收信息）
- 通信协议：团队需要**自创简写**（如"N3"=向北3格，"!"=危险/墙）

## Win condition
- ✅ 探路者在60步内到达出口 = 胜利
- ✅ 建立高效的"团队语言"并获得高分

## Variants / twist
- **迷雾回声**：回声师只能看到探路者周围3格范围，需要多个回声师协作拼凑完整地图
- **噪音干扰**：随机出现的"假回声"（错误信息），团队需要识别和过滤
- **时间竞速**：两队同时进行相同迷宫，比拼谁能更快建立沟通协议并通关
- **反转模式**：探路者能看到迷宫，但只能盲打指令；回声师看不到迷宫，但需要通过探路者的描述来绘制地图并给出建议
