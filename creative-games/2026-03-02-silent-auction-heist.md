# 静默拍卖劫案 (Silent Auction Heist)

- Date: 2026-03-02
- Players: 4-8人
- Time: 20-30分钟
- Genre: 心理博弈 / 身份隐藏 / 轻策略

## Elevator pitch
一场神秘的地下拍卖会上，混入了一群劫匪。他们要通过暗号和暗示来完成盗窃任务，而真正的收藏家则要在喧嚣中找出谁是叛徒。

## Core loop
1. **物品展示阶段** — 每轮展示一件"拍品"（普通/稀有/传奇）
2. **暗号传递阶段** — 所有玩家同时打出0-5的手势数字
3. **结算阶段** — 计算手势总和，决定拍品归属和事件触发
4. **身份揭露阶段** — 特定条件触发后，部分玩家身份可能被暴露

## Rules (high level)

### 角色分配
- **收藏家 (60%)**: 目标是获得最高总价值的拍品
- **劫匪 (30%)**: 目标是让拍品被"盗走"（总和触发特定数字）
- **内鬼 (10%)**: 表面是收藏家，实际是劫匪头目，可查看其他劫匪身份

### 手势含义
- 0 = 放弃本轮
- 1-3 = 正常出价（价值=数字）
- 4 = 可疑加价（价值×2，但引起注意）
- 5 = 盗窃信号（劫匪专用，触发特殊事件）

### 结算规则
- 手势总和 ≤ 10: 最高出价者获得拍品
- 手势总和 = 11-14: 拍品被"盗"，劫匪得分
- 手势总和 ≥ 15: 引发警报，所有人身份受检，最高者暴露身份
- 出现多个5: 劫匪互相识别，可私下交流一次

## Win condition

### 收藏家胜利
- 持有拍品总价值最高
- 成功指认出所有劫匪

### 劫匪胜利
- 累计盗走3件拍品
- 或：让收藏家错误指认3次（引发混乱）

### 内鬼特殊胜利
- 既让劫匪获胜，自己又是收藏家中的最高分
- 独自获胜，背叛所有人

## Variants / twist

### 极速模式
- 每轮限时30秒，增加紧张感
- 手势改为打字输入（线上版）

### 身份反转
- 游戏进行到一半时，所有身份互换
- 收藏家变成劫匪，反之亦然

### 团队模式
- 4v4对战
- 双方各有2名劫匪互相渗透
- 需要判断谁是卧底队友

### 拍卖会大师
- 加入"拍卖师"NPC角色
- 由AI或主持人扮演，操控节奏
- 加入随机事件和突发状况
